Tour & Travel Management System(MilesToFly)

MilesFly, Tour and Travel website application used to provide the complete information about the tour package and agencies available for a tour. Main objective of this system is to provide Online Registration, Tour Package Information, Ticket Booking, Online Payment and Searching Facility for customer and etc.

Admin has the overall control to this website.Main task in Admin modules are approve Agency,add category,view users and Package Details ,View Booking and Payment Details,Send e-mail alerts. Agency can register on site .if admin gives approval, agency can login and can add packages. Agency has control to the package package booking and payment. Registered user has privileges to send the enquires to all agency, give feedback and rating, view all packages and booking etc.

System Requirements

Html,Css,Js,Bootstrap Ajax PHP MySQL Database Google Sign-In API payumoney Gateway
